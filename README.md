# 発災時行動判断支援ツール

東京医療減災Labo（東京都立病院機構）が無償提供する、
病院の発災時行動判断を支援するHTMLツールです。

## 使い方

1. 下の **Releases** からツールをダウンロード
2. ダウンロードした `bcp_disaster.html` をブラウザで開く
3. 「初回設定」タブで病院名・診療科チェックを入力して保存

ネット接続不要。設定はブラウザのローカルストレージに保存されます。

## 設計思想

EMIS等の外部システムとの連携はあえて行いません。
完全に独立した、シンプルで確実に動くツールを目指しています。
利便性の追求は、災害時には障害の原因になります。
シンプル・イズ・ベスト。
